# 📊 Data Sources – TechInterviewAI

Este documento describe las fuentes de datos consideradas para el desarrollo del proyecto **TechInterviewAI: Simulador de Entrevistas Técnicas con IA Generativa**, así como sus características, riesgos y consideraciones éticas.

---

# 1️⃣ Tipo de Datos Utilizados

El proyecto trabaja principalmente con:

- Datos de tipo texto
- Preguntas técnicas
- Respuestas modelo
- Categorías y niveles de dificultad

No se utilizan datos numéricos sensibles ni información personal.

---

# 2️⃣ Fuentes de Información

## 2.1 Documentación Técnica Oficial

Fuentes públicas de documentación tecnológica:

- Documentación oficial de lenguajes de programación
- Documentación de frameworks backend y frontend
- Recursos técnicos abiertos

Tipo de datos:
- Explicaciones técnicas
- Definiciones conceptuales
- Ejemplos de código

Licencia:
- Uso público
- Documentación abierta

---

## 2.2 Repositorios Públicos

Referencias consideradas:

- Repositorios abiertos en GitHub
- Bancos públicos de preguntas técnicas
- Material educativo abierto

Tipo de datos:
- Preguntas de entrevistas técnicas
- Respuestas explicativas

Licencia:
- Código abierto
- Uso académico

---

## 2.3 Generación Sintética

Parte del contenido podrá generarse utilizando modelos de lenguaje (LLM) para crear preguntas adicionales o reformular contenido técnico.

Estas preguntas generadas serán:

- Revisadas manualmente
- Ajustadas a nivel académico
- Utilizadas exclusivamente con fines educativos

---

# 3️⃣ Datos Incluidos en el Repositorio

El repositorio incluirá únicamente:

- Archivos de ejemplo
- Datos sintéticos de prueba
- Estructuras de demostración

No se incluirán datasets completos descargados externamente.

---

# 4️⃣ Datos NO Incluidos

Por razones éticas y académicas, NO se incluyen:

- Datos personales (PII)
- Respuestas reales de candidatos
- Información privada
- Bases de datos comerciales
- Contenido con restricciones de licencia

---

# 5️⃣ Tamaño Estimado del Dataset Conceptual

El dataset conceptual contempla:

- Entre 500 y 2000 preguntas técnicas
- Clasificadas por categoría (Backend, Frontend, Data)
- Clasificadas por nivel (Junior, Semi-Senior, Senior)

Sin embargo, el repositorio solo contendrá muestras reducidas.

---

# 6️⃣ Riesgos Identificados

- Sesgo hacia tecnologías populares.
- Información potencialmente desactualizada.
- Alucinaciones del modelo generativo.
- Evaluación automática imperfecta.
- Dependencia de la calidad del prompt engineering.

---

# 7️⃣ Consideraciones Éticas

TechInterviewAI:

- No recolecta información personal.
- No almacena respuestas reales de usuarios.
- No realiza perfilamiento.
- Se desarrolla exclusivamente con fines académicos.

Se prioriza la transparencia y el respeto por las licencias abiertas.

---
